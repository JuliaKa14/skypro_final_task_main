Git

  • git init — используется для инициализации нового репозитория Git в текущей директории проекта или для преобразования существующего проекта в репозитории
  • git clone — используется для создания локальной копии существующего удаленного репозитория Git
  • git add — используется для добавления изменений из рабочего каталога в индекс для последующего коммита
  • git commit — команда для сохранения изменений в локальном репозитории Git
  • git push — отправляет локальные коммиты в удалённый репозиторий, синхронизируя изменения между локальной и удалённой версиями проекта
  • git pull — загружает изменения из удаленного репозитория и сразу же объединяет их с текущей локальной веткой
  • git status — отображает текущее состояние рабочего каталога и области индекса репозитория
  • git log — показывает историю коммитов репозитория, отображая информацию о каждом из них: хэш, автора, дату и сообщение
