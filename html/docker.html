Docker

Основные команды Docker

  • docker --version — показать версию Docker CLI (интерфейса командной строки)
  • docker images — вывод списка всех образов Docker, которые доступны локально
  • docker ps — список запущенных контейнеров
  • docker ps -a — список всех контейнеров
  • docker run — запускает новый контейнер из образа Docker, создавая и запуская его одновременно
  • docker stop — используется для корректной остановки одного или нескольких работающих контейнеров Docker
  • docker rm — используется для удаления одного или нескольких Docker-контейнеров
  • docker rmi — удаление образа с локального хоста
  • docker build — создание образов на основе инструкций из файла
  • docker pull — загрузка образа из реестра на локальный компьютер

Полезные команды

  • docker logs — просмотр логов контейнера
  • docker exec — выполнение новой команды внутри работающего Docker-контейнера
  • docker-compose up — запуск многоконтейнерного приложения
  • docker-compose down — остановка многоконтейнерного приложения
